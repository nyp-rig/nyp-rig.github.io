# NYP Robotics CCA Website
By Azlan and Lucas.<br>
Azlan did `index.html` and `programming.html`, while Lucas did `building.html`, `join-us.html`, and `timetable.html`. However, we also interchangably used each others' components and layouts. For example, Azlan's landing page layout and navigation bar is used throughout the websites' pages, while Lucas's Carousel is also used in Azlan's `programming.html`.<br>
**NOTE:** [Node JS](https://nodejs.org) is required to package and compile the SCSS. SCSS is CSS but with some extra features, based on consultation we've been told that we are allowed to use it.
## Installing
```
npm install
```

## Building
```
npm run build
```
Output files will be at `./dist`

## Development/Testing/Running The Project
```
npm run dev
```

## Use of AI in Project
- We have used AI to assist in writing some text content for the site such as descriptions for events.<br>
- `/assets/images/bg.png` was an image generated with A.I. using the [Animagine XL 4.0](https://huggingface.co/cagliostrolab/animagine-xl-4.0) model.
